# SubPilot

Application de gestion d'abonnements simple et élégante.

## Installation

```bash
git clone https://github.com/anastas-hub/SubPilot.git
cd SubPilot
npm install
```

## Utilisation

- Développement :
  ```bash
  npm run start
  ```
- Build (installeur) :
  ```bash
  npm run rebuild
  npm run build
  ```

## Fonctionnalités
- Ajout, édition, suppression d’abonnements
- Dashboard avec totaux
- Interface moderne
